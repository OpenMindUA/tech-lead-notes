# Open Guide to Kanban — distilled

Cards distilled from [The Open Guide to Kanban v2025.7](https://kanbanguides.org/open-guide-to-kanban/) — a community-expanded adaptation of *The Kanban Guide*. The Open Guide preserves the original strictly and **adds** material in italics: more definitions, an expanded metrics catalog, an explicit Outcomes/Impact/Value section, and an appendix on prioritization and WIP control techniques.

> **License:** CC BY-SA 4.0. These cards are derivative; ShareAlike inherits.

> Read the canonical [`distilled/kanban-guide/`](../kanban-guide/index.md) cards first. The Open Guide repeats everything in *The Kanban Guide* and only the **deltas** are distilled here.
