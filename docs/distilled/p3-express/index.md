# P3.express manual (v2) — distilled

Cards distilled from the canonical [P3.express manual v2](https://p3.express/manual/v2/) — an open, minimalist project-management methodology built to comply with the [NUPP principles](../principles/nupp/index.md). 33 management activities in 7 groups; 4 documents; designed to be **used as-is** with **gradual** tailoring rather than upfront design.
