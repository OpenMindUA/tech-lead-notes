# Distilled cards (Tier 2)

This is a dense, agent-friendly distillation of Nader K. Rad's
*PMBOK Guide: Underneath the Surface* (CC-BY-4.0). Anecdotes,
historical commentary and personal asides are removed. Each card is a
self-contained brief with a fixed shape:

- **TL;DR** — one or two sentences of essence
- **Core ideas** — the actionable points
- **When this applies** — decision contexts
- **Anti-patterns** — failure modes the original text warns against
- **Cross-refs** — related concepts inside this book
- **Source** — link back to the full original chunk

Use these cards for retrieval-augmented generation, agent context, or
human "fast read" of a topic. For verbatim quotes, citation, or
nuance, follow the **Source** link to the full chunk.
