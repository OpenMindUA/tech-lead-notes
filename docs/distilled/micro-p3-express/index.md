# micro.P3.express — distilled

Cards distilled from the [micro.P3.express manual](https://micro.p3.express/) — a flavor of [P3.express](../p3-express/index.md) for **micro-projects with 1-7 team members**.

> **License:** CC BY 4.0. These cards are derivative; same license applies.

## What's different from P3.express

micro.P3.express is **not a separate methodology** — it's a stripped-down variant. The most consequential differences:

- **Roles → "hats"** — same person may wear several; concerns matter more than authorities.
- **No Monthly cycle** — only Weekly (C+E groups merge cycle responsibilities). Monthly cycle is preserved as an option for *stretched* micro-projects.
- **Tooling is part of A3** — picking tools and creating a project repository is an explicit activity, because micro teams often start without infra.
- **Compact: 27 activities** in 6 groups (vs P3.express's 33 in 7).

For the canonical full P3.express text and broader cards, see [P3.express distilled](../p3-express/index.md).
