# Scrum Guide 2020 — distilled

Cards distilled from the canonical *2020 Scrum Guide* by
Ken Schwaber and Jeff Sutherland, focused on **what an agent or
practitioner needs**: prescriptions, accountabilities, anti-patterns
and cross-refs. The Scrum Guide itself is short and prescriptive;
cards add anti-patterns and cross-source references rather than
re-stating.
