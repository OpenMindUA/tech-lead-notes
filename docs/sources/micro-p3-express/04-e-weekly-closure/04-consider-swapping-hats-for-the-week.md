# E4 - Consider swapping hats for the week

  

This is the end of the *Weekly Closure* activity group, and we have one more thing to do.

Project Manager Hat

  

Can we increase collaboration by swapping the hats for next week?

It’s usually helpful to redistribute the hats to increase team members involvement and collaboration and improve everyone’s understanding of the project. However, it’s not always possible or desirable.

You need to add the new hat assignments to the “stakeholders” meta-card in the “project description” column of the *Integrated Project Board*.
